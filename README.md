# cursor-dance

## Features

| Feat                   | shortcut       |
| ---------------------- | -------------- |
| SwapSelectionCursor    | ctrl+shift+o   |
| MoveCursorLineToCenter | ctrl+shift+c c |
| MoveCursorLineToTop    | ctrl+shift+c t |
| MoveCursorLineToBottom | ctrl+shift+c b |
